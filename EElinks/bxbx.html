Bxbx

Introduction

Bxbx is a simple java based game that I am currently developing. The game follows the unlikely hero Bxbx, a blue box with a small body and big dreams. You follow Bxbx as he moves forward and help him avoid obstacles and collect tokens. I am a single developer in this game and am always looking for more help!

Programming Development

With this game, it was my goal to learn about how to use Java as I am more familiar with Python and C++. I have made several iterations of the code in order to optomize the preformance of the game. My first iteration of the game was done completely using a single class. The game ended up having difficulties loading certain assests. I am now working on reformating the game into appropriatly.
